VOXEL 3D

Introduce 3D Voxel Models

Web application for converting 3D models into voxel models that can convert 3D models into voxel models with texture colors based on the color of input models into formats. Web application intended to develop a web application for converting 3D models into voxel models for use in platforms such as game assets and non-fungible tokens (NFTs).
